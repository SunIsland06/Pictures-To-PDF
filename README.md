# Pictures-To-PDF （施工中）

## 功能描述

将图片一键转换为PDF文件。支持选择单个文件夹或多个图片文件（JPG、PNG、JPEG格式），快速生成PDF文档。

## 主要特性

- 支持批量处理多个图片文件
- 支持选择整个文件夹进行转换
- 支持JPG、PNG、JPEG等多种图片格式
- 一键生成PDF文件
- 保持图片原始质量和比例
